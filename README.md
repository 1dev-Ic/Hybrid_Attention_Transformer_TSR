# Hybrid_Attention_Transformer_TSR
Evaluation of Attention-Augmented and Hybrid Transformer Architectures for Real-Time Traffic Sign Recognition in Intelligent Transportation Systems

## Project Overview

This repository contains the final project for **AIE 510 – DEEP LEARNING**. The project evaluates attention-augmented and hybrid transformer-based deep learning architectures for real-time traffic sign recognition in Intelligent Transportation Systems.

Traffic sign recognition is an important computer vision task in autonomous driving and smart transportation systems. Accurate and efficient recognition of traffic signs can support driver assistance systems, improve road safety, and contribute to the development of intelligent mobility solutions.

The study compares different deep learning architectures designed to balance classification performance, computational efficiency, and real-time inference capability.

The project evaluates attention-based and hybrid deep learning models for traffic sign recognition, including:

CNN + Vision Transformer
EfficientNet + Squeeze-and-Excitation
ResNet + LSTM

These models were selected to compare the effectiveness of convolutional feature extraction, attention mechanisms, transformer-based representation learning, and sequence modeling.
